# 登录界面部分

## 验证码部分（login.vue文件）

- 定义了一个时间倒计时功能，运用了防抖和时间倒计时功能，同时检测手机号格式是否正确

  - ~~~js
    // 发送验证码，进行倒计时
    const countdown = reactive({
      validText: '获取验证码',
      time: 60,
    })
    
    // 定义flag变量，检测用户是否点击了获取验证码
    let flag = false
    
    const countdownChange = () => {
      // 防止重复点击
      if (flag) {
        return
      }
    
      //判断手机号格式是否正确（正则表达式）
      const phoneReg = /^1(3[0-9]|4[01456879]|5[0-35-9]|6[2567]|7[0-8]|8[0-9]|9[0-35-9])\d{8}$/
      if (!loginForm.userName || !phoneReg.test(loginForm.userName)) {
        return ElMessage({
          message: '请输入正确的手机号',
          type: 'warning',
        })
      }
    
      // 倒计时，定时器
      setInterval(() => {
        if (countdown.time <= 0) {
          countdown.validText = '获取验证码'
          // 完成倒计时，清除定时器
          clearInterval(intervalID);
          // 倒计时结束之后，重置flag
          flag = false
        } else {
          countdown.time -= 1
          countdown.validText = `剩余${countdown.time}秒`
        }
      }, 1000)
      flag = true
    }
    ~~~

##　表单的校验规则（在标签中调用:rules="rules"）

- 账号和密码的检验规则

  - ~~~js
    // 账号校验规则
    const validateUser = (rule, value, callback) => {
      // 账号不为空
      if (value === '') {
        callback(new Error('请输入账号'))
      } else {
          // 校验手机号的正则
        const phoneReg = /^1(3[0-9]|4[01456879]|5[0-35-9]|6[2567]|7[0-8]|8[0-9]|9[0-35-9])\d{8}$/
        phoneReg.test(value) ? callback() : callback(new Error('手机号格式不对,请输入正确的手机号'))
      }
    }
    // 密码校验规则
    const validatePass = (rule, value, callback) => {
      // 密码不为空
      if (value === '') {
        callback(new Error('请输入密码'))
      } else {
          // 校验手机号的正则
        const reg = /^[a-zA-Z0-9_-]{4,16}$/
        reg.test(value) ? callback() : callback(new Error('密码格式不对,请输入正确的密码'))
      }
    }
    // 表单校验规则
    const rules = reactive({
      userName: [
        {
          validator: validateUser,
          // 失去焦点时触发校验，触发的时候会调上一行的函数
          trigger: 'blur'
        }
      ],
      passWord: [
        {
          validator: validatePass,
          // 失去焦点时触发校验，触发的时候会调上一行的函数
          trigger: 'blur'
        }
      ]
    })
    ~~~

## 添加了路由守卫(main.js)

~~~js

// 前置导航守卫(to为跳转的路径，from为从哪里跳转的)
router.beforeEach(( to,from) => {
    // 这里可以做一些全局的前置操作，比如判断是否登录，是否有权限等
  localStorage.getItem('pz_token')
  // 非登录页面跳转，如果token不存在且不在登录页面，则跳转到登录页面
  if (!token && to.path !== '/login') {
      return '/login'
  } else if (token && to.path === '/login') { 
      return '/'
  } else {
    return true
  }
})
~~~
